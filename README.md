# AI English Learning Assistant (RAG System)

Програмний комплекс для інтелектуальної підтримки вивчення англійської мови на основі архітектури **RAG (Retrieval-Augmented Generation)**. Проєкт розроблений у межах навчального курсу з основ штучного інтелекту.

## Опис проєкту
Застосунок дозволяє користувачу завантажувати навчальні PDF-мануали або підручники та взаємодіяти з ними через чат-інтерфейс. Система використовує векторний пошук для знаходження релевантних частин тексту та передає їх моделі **Gemini API** для генерації точних відповідей, що базуються виключно на завантажених матеріалах.

## Основні характеристики
* **Векторна база даних**: Використання FAISS для швидкого пошуку за подібністю.
* **Текстовий розщеплювач**: RecursiveCharacterTextSplitter для обробки великих документів без втрати контексту.
* **Безпека**: Використання змінних оточення (.env) для захисту API-ключів.
* **Захист від галюцинацій**: Суворі системні інструкції, що обмежують відповіді ШІ лише темою англійської мови та наданим контекстом.

## Технологічний стек
* **Мова**: Python
* **ШІ Модель**: Google Gemini 1.5 Flash / Pro
* **Ембеддінги**: Google Generative AI Embeddings
* **Фреймворки**: LangChain, Streamlit
* **Векторне сховище**: FAISS
